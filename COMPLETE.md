## 🚀 To Run Right Now

### 1. Backend
```bash
cd backend
npm install
# Create .env file
npm run dev
```

### 2. Frontend
```bash
cd frontend
npm install
npm start
```

### 3. Choose Platform
- Press `w` → **Web browser**
- Press `a` → **Android** (Expo Go)
- Press `i` → **iOS** (Expo Go)

## 🌐 Deploy Web

```bash
cd frontend
npm run build:web
# Deploy 'web' folder to Vercel/Netlify/Firebase
```

## 📱 Deploy Mobile

```bash
# Install EAS CLI
npm install -g eas-cli

# Build
eas build --platform android  # Android
eas build --platform ios      # iOS
```
